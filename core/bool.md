# 불리언 

## 1. 불리언 값 다루기

```python
type(True)

isinstance(True, int)

True == 1

True is 1

True or False

True + 4
```

## 2. 불리언 값의 조건절 활용


```python
a = 1
if a == 1: print(a == 1)
```
